## Requirements

For building and running application you need:

- Node.js 18.16.0
- Yarn 1.22.19
- Typescript 5.1.3

## Installation

```Installation
$ git clone https://github.com/elice-starRoad/Frontend
$ git clone https://github.com/elice-starRoad/Backend
$ cd starRoad
```

## Backend

```Backend
$ yarn && yarn start
```

## Frontend

```Frontend
$ npm install && npm run dev
```
